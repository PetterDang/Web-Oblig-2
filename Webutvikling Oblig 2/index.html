Git / Github


GitHub Inc er et web-basert service som tar i bruk Git som er et system for distribuert versjonskontroll. I GitHub er alle distribuerte versjonskontroll og SCM funksjoner inkludert. I tilleg har GitHub flere funksjoner. Ved å ha distribuert versjonskontroll gjør det det mulig å se alle endringer gjort på filene. Dette gjør det lettere å samarbeide med andre utvikliere.

Media Typer


Media typer gjør det mulig å bruke CSS til å bestemme regler på stil på forskjellige media typer. Ved å bruke @media kan du bestemme hva slags stil siden skal ha på forskjellige plattformer som datamaskiner, tabletter, mobiler og bærebare datamaskiner.

CSS Layouts


Det er mange forskjellige metoder for layouts i CSS for eksempel ved bruk av CSS Flexbox eller CSS Grid. Det er også andre metoder for å strukturere layouts i CSS som er mindre responsive eller ikke responsive som for eksempel table layout, floats og posisjonering.. Den mest vanlige måten å strukturere en webside er ved å dele dem inn i "header", "meny", "innhold" og "footer"
