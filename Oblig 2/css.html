CSS Layouts

Det er mange forskjellige metoder for layouts i CSS for eksempel ved bruk av CSS Flexbox eller CSS Grid. Det er også andre metoder for å strukturere layouts i CSS som for eksempel table layout, floats og posisjonering.

Med Flex kan innholdet bli omformet etter enheten som blir brukt. Dette betyr at ved bruk av flex blir siden justert etter skjerments størrelse. Innholdet i flex flytter seg automatisk basert på hvor mye plass det er på skjermen. Innholdet vil heller ikke legge seg oppe på hverandre men heller legge seg under hverandre for å få plass.

Med Grid blir siden delt inn i rader og kolonner. Du kan selv velge selv hvor mange kolonner og rader det er. Fra her kan du velge størrelse på boksene og mellomrom mellom dem. Boksene blir delt inn i tema som danner en vanlig side struktur.

Det er mulig å kombinere flex og grid sammen. Dette tillater brukeren til å bruke de beste egenskapene fra funksjonene som grid og flex tilbyr. Det er mange måter å strukturere en webside på. Den mest vanlige måten å strukturere en webside er ved å dele dem inn i "header", "meny", "innhold" og "footer"
