Media Typer

Media typer gjør det mulig å bruke CSS til å bestemme regler på stil på forskjellige media typer. Ved å bruke @media kan du bestemme hva slags stil siden skal ha på forskjellige plattformer som datamaskiner, tabletter, mobiler og bærbare datamaskiner.

Dette blir brukt til å velge blandt annet bredde og høyde av synsfeltdimensjonen eller type enhet, hvilken orientasjon enheten er i som landskaps modus eller portrett modus og resolusjon. Du kan gjøre alt dette ved å lage et nytt CSS ark eller putte en kode i CSS arket ditt. Dette blir brukt til å lage resoponsive nettsider. Dette kan også bli brukt til å spesifisere hvordan produktet skal se ut hvis det blir printet ut.

Det er fire media typer som blir mest brukt. "All" som spesifiserer at alle media enhetene bruker denne stilen. "Print" som er resultatet av filen hvis den blir printet ut. "Screen" som blir brukt for datamaskiner, tabletter, smart telefoner og lignende. Siste er "Speech" som er brukt for skjermlesere som leser det som står på skjermen ut høyt. Gamle Media typer som "Tv" og "Projection" er sjeldent brukt lenger.
